# AER201

## Pin Assignment

### ADC
- RA0: Tap Conductivity Sensor
- RA1: Side Conductivity Sensor 
- RA2: Light Sensor 

### Servo 
#### Non-Continuous
- RA3: Side Servo, the one on the bottom platform
- REO: Main Servo, the one on the top platform
#### Continuous
- RC2: Gate

### Motor
- RC1: Blocker
- RB0: Shaker
- RD1: Launcher

### Solonoid
- RD0: Platform Solonoid
